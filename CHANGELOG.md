# Changelog
All notable changes to this project will be documented in this file.

## [0.2.0] - 2019-08-02
### Added
- Toolbar support.
- Texture updating.
- `texture_coords()`.
- Culling & Depth test.
- Camera, more `Transform` functions.
- Keyboard input.
- Texture atlas support with custom mipmaps.

### Fixed
- Shape builder's `face()` not working correctly.

## [0.1.0] - 2019-07-07
